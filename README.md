# Brick Breaker

A Java implementation of the classic Brick Breaker (Arkanoid) game developed as part of an Object-Oriented Programming course.

## Features

- Multiple game levels
- Ball physics
- Collision detection
- Keyboard controls
- Score tracking
- Object-Oriented Programming design

## Technologies

- Java
- Object-Oriented Programming

## Running the project

Run `Ass6Game.java`.
